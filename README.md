# Big Cart - Առաքման Հավելված

**Big Cart**-ը բջջային հավելված է առաքման համար, կառուցված React Native-ի վրա և ինտեգրված Platzi Fake API-ի հետ։ 

---

## Տեխնոլոգիական Սթեկ

- **Հիմնական Շրջանակ:** React Native  
- **API:** [Platzi Fake API](https://fakeapi.platzi.com/)  
- **Նավիգացիա:** React Navigation  
- **UI/UX:** Համապատասխանում է Figma դիզայնին՝ գույներ, տառատեսակներ, spacing  

---

## API Էնդփոյնթներ

| Ֆունկցիոնալություն       | Էնդփոյնթ                     | Մեթոդ | Նկարագրություն |
|--------------------------|--------------------------------|--------|----------------|
| Կատեգորիաներ             | `/api/v1/categories`           | GET    | Ստանալ բոլոր կատեգորիաները |
| Ապրանքների Ցանկ          | `/api/v1/products`             | GET    | Ստանալ բոլոր ապրանքները, ֆիլտրի համար |
| Մուտք (Login)             | `/api/v1/auth/login`           | POST   | Օգտատիրոջ մուտք (թեստային տվյալներով) |
| Օգտատիրոջ Պրոֆիլ       | `/api/v1/users/{id}`           | GET    | Ստանալ ընթացիկ օգտատիրոջ տվյալները |
| Նոր Պրոդուկտ (օպցիոնալ) | `/api/v1/products/`            | POST   | Ապրանքի ստեղծում (թեստային նպատակով) |

---

## Հիմնական Էկրաններ

1. **Home Screen**  
   - Կատեգորիաների ցուցադրում (հորիզոնական, ոլորվող)  
   - Առաջարկվող ապրանքներ  
   - Search Bar → որոնման էկրան  

2. **Product Listing Screen**  
   - Ֆիլտրացիա ըստ կատեգորիայի  
   - Ապրանքի քարտեր՝ նկար, անուն, գին, Add to Cart կոճակ, ֆավորիտ  

3. **Product Details Screen**  
   - Մեծ նկար, ամբողջական նկարագրություն, գին  
   - Զամբյուղի և ֆավորիտի ինտեգրում  

4. **Cart Screen**  
   - Զամբյուղի կառավարում (տեղական վիճակ)  
   - Ավելացնել/Նվազեցնել քանակը  
   - Հեռացնել ապրանքը  
   - Հաշվարկել ընդհանուր գումարը (ներառյալ առաքումը)  
   - Checkout կոճակ → հաջողության հաղորդագրություն  

5. **Profile Screen**  
   - Մուտք/Ելք  
   - Օգտատիրոջ անուն և էլ. փոստ՝ `/api/v1/users/{id}`  

6. **Favorites Screen**  
   - Ֆավորիտների կառավարում (տեղական վիճակ)  
   - Ապրանքի հեռացում կամ ավելացում ֆավորիտներից  
   - Ապրանքի անմիջական ավելացում զամբյուղ
